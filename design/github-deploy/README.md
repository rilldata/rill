# Rill Publish — GitHub Deploy UI Prototype

Static HTML/CSS/JS prototype of the Rill publish and GitHub deploy workflow.

## How to view

Open `rill-publish-v2.html` in a browser — no build step needed.

## UI states covered

| File | Description |
|------|-------------|
| `rill-publish-v2.html` | Main prototype — all publish/deploy states |
| `index.html` | Dashboard home |
| `deploy.html` | Deploy overview |
| `branches.html` | Branch management |
| `history.html` | Commit history |
| `dashboard.html` | Analytics dashboard |
| `app.js` | Shared interactivity (dropdowns, modals, tabs, toasts) |
| `styles.css` | Shared design tokens and component styles |
