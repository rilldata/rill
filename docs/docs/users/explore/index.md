# Explore Your Data in Rill

Once you've built your data pipeline, Rill provides powerful tools to explore, analyze, and share your insights.

### AI-Powered Analysis
- **[AI Chat in Rill Cloud](/users/explore/ai-chat)** - Have conversations with your data using AI-powered chat interface
- **[Natural Language Queries (MCP)](/users/explore/mcp)** - Ask questions about your data using plain English with Claude integration

### Interactive Dashboards

- **[Slice-and-Dice Explore Dashboards](/users/explore/dashboard-101)** - Interactive dashboards for dynamic data exploration and filtering
- **[Multiple Measures in Dimension Leaderboard](/users/explore/dashboard-101/multi-metrics)** - Easily compare multiple measures in the dimension leaderboard
- **[Time Dimension Details](/users/explore/dashboard-101/tdd)** - Drill down into time-based analysis
- **[Pivot Tables](/users/explore/dashboard-101/pivot)** - Create cross-tabulations and data summaries
- **[Filtering](/users/explore/filters)** - Apply filters to focus on specific data subsets
<!-- - **[Traditional Canvas Dashboards](/users/explore/canvas-dashboards)** - Custom dashboard layouts with Vega-Lite visualizations -->

### Cloud Features
- **[Bookmarks & Sharing](/users/explore/bookmarks)** - Save and share specific dashboard views and filters
- **[Share with External Users](/users/explore/public-url)** - Invite external users and control their access to your dashboards
- **[Exports and Scheduled Reports](/users/explore/exports)** - Export data and set up automated report delivery
- **[Alerts](/users/explore/alerts)** - Set up notifications when data meets certain conditions
