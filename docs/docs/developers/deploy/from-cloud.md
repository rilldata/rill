## Overview

Rill Cloud editing lets you create and modify projects entirely in the browser. Changes are deployed automatically as you save, with no need to push from a local machine or manage a Git repository.

This workflow is ideal for:

- **Business users** who want to create or adjust dashboards without setting up a local environment
- **Quick edits** to production projects without cloning the repo
- **Teams** that prefer a fully hosted workflow
