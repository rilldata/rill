## Overview

Connect your ClickHouse OLAP engine to Apache Hudi tables using ClickHouse's [`hudi()` table function](https://clickhouse.com/docs/en/sql-reference/table-functions/hudi).

## Model Configuration

### Reading from S3

Create `models/hudi_data.yaml`:

```yaml
type: model
connector: my_clickhouse

sql: |
  SELECT *
  FROM hudi(
    'https://my-bucket.s3.amazonaws.com/path/to/hudi/table',
    '{{ .env.connector.s3.aws_access_key_id }}',
    '{{ .env.connector.s3.aws_secret_access_key }}'
  )
```
