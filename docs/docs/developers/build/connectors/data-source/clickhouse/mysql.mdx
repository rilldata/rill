## Overview

[MySQL](https://www.mysql.com/) is one of the most widely used relational databases. ClickHouse can query MySQL tables directly using its [`mysql()` table function](https://clickhouse.com/docs/en/sql-reference/table-functions/mysql), pulling data into your Rill models without intermediate exports.

<ClickHousePrereq />

## Credentials

Add your MySQL credentials to your project's `.env` file:

```bash
MYSQL_HOST=your_host
MYSQL_PORT=3306
MYSQL_DATABASE=your_database
MYSQL_USER=your_user
MYSQL_PASSWORD=your_password
```

For details on managing credentials, see [Configure Local Credentials](/developers/build/connectors/credentials).

## Model Configuration

```yaml
type: model
connector: my_clickhouse

sql: |
  SELECT *
  FROM mysql(
    '{{ .env.MYSQL_HOST }}:{{ .env.MYSQL_PORT }}',
    '{{ .env.MYSQL_DATABASE }}',
    'my_table',
    '{{ .env.MYSQL_USER }}',
    '{{ .env.MYSQL_PASSWORD }}'
  )
```

:::tip Network Access
Your ClickHouse instance must be able to reach the MySQL server. If you're using Rill-Managed ClickHouse, ensure your MySQL database is accessible from the internet or configure appropriate network peering.
:::
