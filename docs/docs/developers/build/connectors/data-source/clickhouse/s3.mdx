## Overview

[Amazon S3](https://docs.aws.amazon.com/AmazonS3/latest/userguide/Welcome.html) is a scalable object storage service from AWS. ClickHouse reads S3 data directly using its [`s3()` table function](https://clickhouse.com/docs/en/sql-reference/table-functions/s3), which supports Parquet, CSV, JSON, and other formats.

<ClickHousePrereq />

## Credentials

Add your AWS credentials to your project's `.env` file:

```bash
AWS_ACCESS_KEY_ID=your_access_key_id
AWS_SECRET_ACCESS_KEY=your_secret_access_key
```

For details on managing credentials, see [Configure Local Credentials](/developers/build/connectors/credentials).

## Model Configuration

```yaml
type: model
connector: my_clickhouse

sql: |
  SELECT *
  FROM s3(
    'https://my-bucket.s3.amazonaws.com/path/to/data/*.parquet',
    '{{ .env.AWS_ACCESS_KEY_ID }}',
    '{{ .env.AWS_SECRET_ACCESS_KEY }}',
    'Parquet'
  )
```

:::info
ClickHouse uses HTTPS-style URLs (`https://my-bucket.s3.amazonaws.com/...`) rather than `s3://` URIs.
:::

## Path Patterns

Use wildcards and globs to read multiple files:

```sql
-- Single file
SELECT * FROM s3('https://my-bucket.s3.amazonaws.com/data/file.parquet', ...)

-- All Parquet files in a directory
SELECT * FROM s3('https://my-bucket.s3.amazonaws.com/data/*.parquet', ...)

-- Specific partitions
SELECT * FROM s3('https://my-bucket.s3.amazonaws.com/data/{2024,2025}/*.parquet', ...)

-- Numbered ranges
SELECT * FROM s3('https://my-bucket.s3.amazonaws.com/data/part_{1..10}.parquet', ...)
```

## Supported Formats

The last argument to `s3()` specifies the file format. Common options include `Parquet`, `CSVWithNames`, `JSONEachRow`, and `TSVWithNames`. See the [ClickHouse formats documentation](https://clickhouse.com/docs/en/interfaces/formats) for the full list.

## S3-Compatible Storage

You can also use `s3()` with S3-compatible services (MinIO, DigitalOcean Spaces, etc.) by specifying the service's endpoint URL:

```yaml
sql: |
  SELECT *
  FROM s3(
    'https://my-space.nyc3.digitaloceanspaces.com/path/to/data.parquet',
    '{{ .env.S3_ACCESS_KEY_ID }}',
    '{{ .env.S3_SECRET_ACCESS_KEY }}',
    'Parquet'
  )
```
